# React Book Review App

## Features
- Login and registration forms
- Authentication
- Navigation bar
- Dynamic data via REST API
- Components, Routing, Hooks, Forms

## Setup
1. Install dependencies:
```bash
npm install
```

2. Run the app:
```bash
npm run dev
```

3. Start JSON server:
```bash
npx json-server --watch db.json --port 3001
```
